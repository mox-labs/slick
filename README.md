# slickit

Semantic, LLM-Interpretable Component Kit.

## What

Foundation types for composable components. Three types:

- **`Manifest`** — what a component is, needs, and produces
- **`TypedStruct`** — typed data envelope (`type_url` + JSON value)
- **`TypedRegistry`** — `type_url` → factory → instance

## Manifest

```yaml
type_url: cix.capability.v1.Recon
source: git+https://github.com/org/recon
requires: [cix.v1.Target]
provides: [cix.v1.ReconReport]
relations:
  skills: [git+https://github.com/org/skills/recon]
```

Five fields. Manifests are generated by LLMs, not written by hand.

## Install

```bash
cargo add slickit --features manifest  # Rust
pip install slickit                     # Python
```
